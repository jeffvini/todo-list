# 📝 Lista de Tarefas - React + TypeScript

Uma Lista de Tarefas moderna e de alta performance desenvolvida para dominar o gerenciamento de estados, componentização e as novas funcionalidades do **Tailwind CSS v4**.

## 🔗 Demonstração ao Vivo

Confira o projeto em produção: [https://jeff-todo-list.vercel.app/](https://jeff-todo-list.vercel.app/)

## 🚀 Tecnologias

- **Framework:** [React 19](https://react.dev/)
- **Ferramenta de Build:** [Vite](https://vitejs.dev/)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
- **Estilização:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Deploy:** [Vercel](https://vercel.com/)

## 🎯 Objetivos do Projeto

O foco principal deste projeto é praticar:

- **State Lifting**: Gerenciamento de estado global a partir do componente pai (`App.tsx`).
- **Componentização**: Divisão da interface em componentes reutilizáveis e especializados.
- **Filtragem Avançada**: Implementação de busca, filtragem por categoria e ordenação alfabética (A-Z/Z-A).
- **Tailwind v4**: Utilização do novo motor para uma estilização mais rápida e limpa.

## 🎨 Créditos e Inspiração

A base visual deste projeto foi inspirada no tutorial do **Matheus Battisti** neste [vídeo](https://www.youtube.com/watch?v=YVEVrigByKY&list=WL&index=3&t=1456s).

> **Nota**: Embora a interface seja baseada no tutorial, toda a lógica, integração com TypeScript e a implementação do Tailwind v4 foram desenvolvidas por mim do zero.
